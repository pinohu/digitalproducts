# 00 · Foundation

Strategic spine. Everything downstream — every product, every sales page, every email — has to be consistent with what's defined here. If a product can't be reconciled with the foundation, the foundation wins; the product doesn't ship.

## Files in this folder

- [`brand-positioning.md`](./brand-positioning.md) — One-sentence positioning, anti-positioning, voice, tone.
- [`audience-profile.md`](./audience-profile.md) — Who we sell to. Avatar, pains, language, watering holes.
- [`value-ladder.md`](./value-ladder.md) — The full price ladder from free to $5K+, with rung-by-rung logic.
- [`pricing-philosophy.md`](./pricing-philosophy.md) — How we set prices, why we never race to the bottom.

## When to update

- Quarterly review (default cadence)
- When a product-market fit signal forces a positioning shift
- When the audience profile drifts (most common after a launch with surprising buyer demographics)
