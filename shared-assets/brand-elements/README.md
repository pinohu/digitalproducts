# Brand Elements

Logos, colors, fonts, voice. Single source of truth for brand consistency across products.

## Contents (Add as Created)

- `logo/` — primary logo, alt logos, favicon, social profile sizes
- `colors.md` — hex codes for primary, secondary, accent, neutral palette
- `fonts.md` — typeface names, hierarchy rules, fallbacks
- `voice-and-tone.md` — voice guide (refer also to `/00-foundation/brand-positioning.md`)
- `cover-templates/` — reusable cover art templates for product PDFs

## Usage

Every product cover, sales page, and email should be checkable against this folder. If a design element doesn't appear here, it shouldn't appear in the product.
