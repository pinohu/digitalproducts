# Swipe Files

Annotated examples of high-converting copy from other products (ours or others'). Each file should explain *why* the copy works, not just preserve it.

## File Pattern

`<source>-<topic>.md` — e.g., `welsh-day-zero-launch-email.md`, `gumroad-cart-abandonment-email.md`

## Inside Each File

```
# [Source / context]

## What it is
[Where this copy was used, who wrote it, when]

## The copy
[The exact copy, verbatim]

## What makes it work
[3–5 specific observations: structure, language, hooks, etc.]

## How we'd adapt it
[What we'd change for our voice / avatar]
```

## Don't Confuse Swipe With Plagiarism

Swipe files are for *learning patterns*, not for copy-paste. Adapt the structure; never lift the language.
