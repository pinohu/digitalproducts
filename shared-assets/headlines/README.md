# Headlines Bank

Tested headlines, with conversion data where available. The pattern bank for headline construction.

## Format

```
## [Headline]

- **Product / context:** Where this was used
- **Conversion rate:** X% (sales page conversion)
- **Pattern type:** From `/04-sales-pages/headline-formulas.md`
- **Why it worked / didn't:** 1–2 sentences
```

## Goal

After 5+ launches, this file becomes a curated bank of headlines that have actually been tested. New product launches start by referencing this file, not from scratch.

## Quarterly Review

Look at all headlines tested in the last 90 days. Identify the top 3 by conversion rate. Promote them to the top of this file as "go-to patterns."
