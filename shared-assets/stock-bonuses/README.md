# Stock Bonuses

Pre-built bonuses that can be plugged into multiple product offers. Save production time when building each new offer.

## What Belongs Here

- Curated tool lists (e.g., "The 50 SaaS Tools I Use" — applies across all our products)
- Generic checklists (e.g., "The Pre-Launch Compliance Checklist")
- Reusable templates (Notion templates, spreadsheets, prompt libraries)
- Generic frameworks (decision flowcharts, scoring rubrics)

## What Doesn't Belong Here

- Product-specific bonuses (those live in `/03-products/<slug>/bonuses/`)
- Anything that wouldn't make sense bundled with multiple unrelated products

## Per-Bonus File

Each bonus has:
- The deliverable file (PDF, Notion link, spreadsheet, etc.)
- A `README.md` with: title, description, standalone value $, where it's been used, performance notes
