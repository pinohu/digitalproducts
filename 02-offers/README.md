# 02 · Offers

The product is not the offer. The **offer** is the entire wrapper: product + bonuses + guarantee + framing + price + risk reversal. Hormozi's full thesis is that the same product can produce 100x different revenue based purely on how the offer is constructed. This folder is where we engineer that wrapper.

## Files in this folder

- [`grand-slam-offer-template.md`](./grand-slam-offer-template.md) — The 4-pillar offer construction worksheet.
- [`value-equation-worksheet.md`](./value-equation-worksheet.md) — Hormozi's Value Equation applied to the specific product.
- [`bonus-stack-template.md`](./bonus-stack-template.md) — How to layer bonuses to lift perceived value 3–10x.
- [`guarantee-templates.md`](./guarantee-templates.md) — Risk reversal language ready to drop into any sales page.

## The Sequence

1. Run the Value Equation worksheet — what's the dream outcome, the perceived likelihood, the time delay, the effort?
2. Engineer the four pillars (Grand Slam Offer template) — value prop, premium price, risk reversal, urgency.
3. Stack the bonuses — each bonus addresses an objection or amplifies the dream outcome.
4. Pick the guarantee from the templates and customize.
5. Move to `04-sales-pages/` to translate the offer into copy.

## The Math

Aim for: **perceived value ≥ 10x price.**

A $49 offer should feel like it's worth $500 to the right buyer. The way to get there isn't to lower the price; it's to raise the perceived value through:

- **Specificity** — naming the exact dream outcome
- **Speed** — collapsing the time delay
- **Certainty** — increasing the perceived likelihood through proof, guarantees, and risk reversal
- **Effort reduction** — making it easier for the buyer to get the result
- **Bonus stacking** — each bonus has its own dollar value visibly attached
